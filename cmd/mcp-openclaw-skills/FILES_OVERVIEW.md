# MCP OpenClaw Skills - Files Overview

This document provides a detailed overview of all files in the MCP OpenClaw Skills plugin.

## Core Files

### `main.go`
**Purpose**: Main server implementation

**Key Components**:
- **Protocol Types**: JSON-RPC request/response structures
- **Skill Types**: Skill metadata and content structures
- **MCPServer**: Main server struct with skills cache
- **Skill Loader**: Parses SKILL.md files and extracts metadata
- **Request Handlers**: Implements MCP protocol methods
- **Tool Implementations**: list_skills, get_skill, search_skills
- **Resource Handlers**: List and read resources

**Key Functions**:
```go
func main()                              // Entry point, initializes server
func (s *MCPServer) loadSkills()         // Loads all skills on startup
func (s *MCPServer) loadSkill()          // Loads individual skill
func parseSkillMetadata()                // Parses YAML frontmatter
func (s *MCPServer) Run()                // Main event loop
func (s *MCPServer) handleRequest()      // Routes MCP requests
func (s *MCPServer) handleInitialize()   // MCP initialization
func (s *MCPServer) handleListTools()    // Lists available tools
func (s *MCPServer) handleCallTool()     // Executes tool calls
func (s *MCPServer) handleListResources() // Lists available resources
func (s *MCPServer) handleReadResource()  // Reads resource content
func (s *MCPServer) listSkills()         // Tool: list all skills
func (s *MCPServer) getSkill()           // Tool: get specific skill
func (s *MCPServer) searchSkills()       // Tool: search skills
```

**Dependencies**:
- `gopkg.in/yaml.v3` - YAML parsing
- Go standard library

**Lines of Code**: ~700
